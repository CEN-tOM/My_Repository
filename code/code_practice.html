MEMO

__innerについて

各セクションの直下の子要素(主にdiv)のclass属性に追加する。
ただしtopやfvには付与しないことが多い。
これを各セクションに付与することで、全てのセクションの左右中央寄せや、
上下のセクション間のpadding設定、widthなどを一括管理できる。
個別にwidthなどの数値を設定したい場合には、
"__inner"と並列に付与している兄弟要素divに個別でwidth指定してあげれば良い。

空白の開け方

キーボードでスペースをあけようとしてもではうまく空白をあけられない。
空白をあけるにはHTMLに決まったコマンドを入力する。

  1. 【&nbsp;】：半角スペースと同じサイズの空白
  2. 【&thinsp;】：&nbsp;の空白より小さい空白
  3. 【&ensp;】：半角スペースより間隔がやや広い空白
  4. 【&emsp;】：全角スペースとほぼ同じサイズの空白

おまけ
【&nbsp;】のような文字をHTMLで文字列として表示させるには
【&】と【nbsp;】の間に【amp;】を入力すると本来空白となる【&nbsp;】を
文字列として表示できる。

セレクタの指定方法について

    いつも忘れてしまうセレクタの指定方法についてまとめる

  1. 【.A B】：クラスA要素より深い階層にある子要素や孫要素のクラスBに効く。
  2. 【.A.B】：クラス名Aとクラス名Bの２つのクラスを持つ要素だけに効く。
  3. 【.A,.B】:クラス名Aの要素とクラス名Bの要素に効く。
